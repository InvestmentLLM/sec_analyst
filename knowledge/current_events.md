# Current Events & Market Context
Add any news, research notes, or market context here.
Claude will automatically include this in every analysis.

## How to use this file
- Paste recent news articles or summaries
- Add sector-specific data or trends
- Add macro economic context (interest rates, inflation, GDP)
- Add company-specific recent news not yet in SEC filings
- This file is loaded fresh on every analysis — no restart needed

## Example entries (delete and replace with real content):
- Fed funds rate is currently X%, with next decision on [date]
- S&P 500 forward P/E is approximately X, above/below historical average of 16x
- [Sector name] sector facing [headwind/tailwind] due to [reason]
